Periodic Table Guide

How elements are organized and what the periodic table reveals about their properties.

Big Idea

The Periodic Table organizes elements by increasing atomic number. Elements in the same group share similar chemical properties because they have the same number of valence electrons.

Key Vocabulary

  • Atomic number — the number of protons; identifies the element
  • Atomic mass — the weighted average mass of an element's isotopes
  • Isotopes — atoms of the same element with different numbers of neutrons
  • Valence electrons — electrons in the outermost energy level; determine how an element bonds
  • Period — a horizontal row on the periodic table; elements in a period have the same number of energy levels
  • Group (Family) — a vertical column; elements in a group have the same number of valence electrons and similar properties

Classification of Elements

🔩

Metals

  • Found on the left side of the periodic table
  • Good conductors of heat and electricity
  • Malleable and ductile
  • Shiny (lustrous)
  • Usually solid at room temperature
💨

Nonmetals

  • Found on the right side of the periodic table
  • Poor conductors (insulators)
  • Brittle as solids
  • Dull appearance
  • Can be solids, liquids, or gases
⚖️

Metalloids

  • Found along the staircase line
  • Have properties of both metals and nonmetals
  • Often used as semiconductors
  • Examples: Silicon, Boron, Germanium

Reading an Element Box

Each element box on the periodic table provides key information:

  • Atomic number (top) — number of protons
  • Element symbol (center) — one or two letter abbreviation
  • Element name — full name of the element
  • Atomic mass (bottom) — weighted average mass in amu

To find the number of neutrons: round the atomic mass to the nearest whole number, then subtract the atomic number.
